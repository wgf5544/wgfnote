在程序执行过程中，重复执行一系列命令的情况是常见的。一个特定的代码块重复执行
直到满足某种条件时终止。如循环遍历某个目录下的所有文件，遍历从控制台输入的参
数等。脚本中的循环语句包括 for 循环语句、while 循环语句和 until 循环语句。
